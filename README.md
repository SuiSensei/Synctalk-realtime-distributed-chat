# Synctalk-realtime-distributed-chat
SyncTalk is a real-time chat application designed to demonstrate how multiple users can communicate instantly using distributed system concepts. It uses WebSocket-based communication to allow seamless message exchange between clients connected to a central server.

**TEST** This project is currently in the testing phase. We are actively working on improving the functionality and user experience. Please feel free to contribute or provide feedback to help us enhance the application.
